This repo contains basic and small web projects and apps I built while learning web development. Projects are intentionally small and focused — perfect for practicing frontend basics (HTML, CSS, JavaScript) and backend fundamentals (Flask, SQL, etc.).

* Practicing frontend and backend web development fundamentals.
* Getting hands-on with responsive design, DOM manipulation, and server-side routing.
* Sharing beginner-friendly web applications and websites.
